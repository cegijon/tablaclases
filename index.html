Elemento Qué es Qué hace Ejemplo Clase de ref. Fecha clase
<html></html> etiqueta Le indica al navegador que es una página web HTML. Se abre arriba de todo y se cierra abajo de todo. <html>contenido de la página web</html> 1 05 agosto 2019
<head></head> etiqueta Le indica al navegador que acá comienza la información que debe tener sobre la página web. Se abre después de y se cierra al terminar el contenido del head. <head>contenido del head</head> 1 05 agosto 2019
<title></title> etiqueta Le indica al navegador el título de la página web (se ve arriba en la barra del navegador). Va dentro del head. <title> Mi primera página web</title> 1 05 agosto 2019
<style type="text/css"> </style> etiqueta (style type) atributo (text/css) Le indica al navegador el estilo de la página web en HTML. Va dentro del head. Incluye los estilos que tienen los diferentes elementos de la página web. <style type="text/css"> h1 {color: red;} p {color: blue;} .verde {color: green;} #titulo {color: orange;} </style> 1 05 agosto 2019
<body></body> etiqueta Encierra el contenido de la página web que ve el usuario. <body><h1>Título</h1><p> Contenido que tendrá el párrafo.</p></body> 1 05 agosto 2019
